# Contributing to spec

This repository follows the organization‑wide contribution guidelines defined in the top‑level [CONTRIBUTING.md](https://github.com/axon-prot/.github/blob/main/profile/CONTRIBUTING.md).

## Repository‑Specific Information

- **Prerequisites**: Git and Node.js (latest LTS) for the conformance runner.
- **Setup**:
  ```bash
  cd conformance/runner
  npm install
  ```
- **Test**: `npm test` (or `npm test -- --endpoint <url>` for running against an implementation).

For the full contribution workflow, code review process, issue handling, and licensing details, please refer to the top-level [CONTRIBUTING.md](https://github.com/axon-prot/.github/blob/main/profile/CONTRIBUTING.md).
